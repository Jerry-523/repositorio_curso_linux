# Exercício 1 — Exemplo

## Objetivo

Listar todos os diretórios e ficheiros no terminal.

### Comando utilizado

```bash
ls
```

### Resultado

![Execução do comando ls](https://github.com/user-attachments/assets/e2c4972c-82f2-4df3-9ebc-a254322ffa84)

### Explicação

O comando `ls` permite listar todos os ficheiros e pastas do diretório atual.

Caso tenha dúvidas sobre a utilização do comando, pode consultar a ajuda executando:

```bash
ls --help
```

Também é possível utilizar o `ls` com diferentes opções para obter informações adicionais, como listar ficheiros ocultos, mostrar detalhes dos ficheiros, entre outras.
